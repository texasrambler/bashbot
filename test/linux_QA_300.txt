File Permissions and Ownership
21. What command changes file permissions?
o chmod (Change Mode)
22. How do you read file permissions in Linux?
o In the format -rwxrwxrwx where first character is file type, and the next 9
characters represent owner, group, and others permissions (read, write,
execute)
23. What do the numbers 754 represent in chmod?
o 7 (rwx=4+2+1) for owner, 5 (r-x=4+0+1) for group, 4 (r--=4+0+0) for others
24. How do you make a file executable?
o chmod +x filename
25. What command changes file ownership?
o chown user filename
26. How do you change both user and group ownership at once?
o chown user:group filename
27. What command sets default permissions for new files?
o umask
28. What is the difference between symbolic and numeric permission notation?
o Symbolic uses letters (r,w,x) and operators (+,-,=)
o Numeric uses octal numbers (0-7)
29. How do you view the permissions in octal format?
o stat -c '%a %n' filename
30. What is the sticky bit and how do you set it?
o It restricts file deletion; set with chmod +t directory or chmod 1xxx directory
31. What does the setuid bit do?
o Allows users to run an executable with the permissions of the file owner; set
with chmod u+s filename
32. What does the setgid bit do?
o When applied to directories, new files inherit the directory's group; set with
chmod g+s directory
33. How do you find all files with specific permissions?
o find /path -perm mode
34. What permissions are required to delete a file?
o Write permission on the directory containing the file
35. What command displays your current user ID and group IDs?
o id
36. How do you change only specific permission bits?
o chmod u+x,g-w,o=r filename
37. How do you copy file permissions between files?
o chmod --reference=reference_file target_file
38. What is ACL in Linux and how do you use it?
o Access Control Lists provide more granular permissions; use setfacl to set and
getfacl to view
39. How do you check if a file has extended attributes?
o lsattr filename
40. What command shows the security context of a file?
o ls -Z filename (on SELinux-enabled systems)
