gst-typefind-1.0 - print Media type of file gst-typefind-1.0 uses the
GStreamer type finding system to determine the relevant GStreamer plugin to
parse or decode file, and the corresponding media type.	 gst-typefind-1.0
accepts the following options: Print help synopsis and available FLAGS
GStreamer info flags to set (list with --help) GStreamer debugging flags to
set (list with --help) GStreamer info and debugging flags to set (list with
--help) Add directories separated with ':' to the plugin search path The
GStreamer team at http://gstreamer.freedesktop.org/
