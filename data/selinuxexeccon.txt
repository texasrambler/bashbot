selinuxexeccon - report SELinux context used for this executable reports the
SELinux process context for the specified command from the specified context
or the current context.
# selinuxexeccon /usr/bin/passwd
staff_u:staff_r:passwd_t:s0-s0:c0.c1023

# selinuxexeccon /usr/sbin/sendmail system_u:system_r:httpd_t:s0
system_u:system_r:system_mail_t:s0
