gettextize - install or upgrade gettext infrastructure [OPTION]... [package-
dir] Prepares a source package to use gettext.	--help print this help and
exit --version print version information and exit -f, --force force writing of
new files even if old exist --po-dir=DIR specify directory with PO files
--no-changelog don't update or create ChangeLog files --symlink make symbolic
links instead of copying files -n, --dry-run print modifications but don't
