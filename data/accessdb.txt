accessdb - dumps the content of a man-db database in a human readable format
will output the data contained within a man-db database in a human readable
form.  By default, it will dump the data from where <db-type> is dependent on
the database library in use.

Supplying an argument to accessdb will override this default.  Print debugging
information.  Print a help message and exit.  Print a short usage message and
exit.  Display version information.
