rmdir - remove empty directories [OPTION]... DIRECTORY...  Remove the
DIRECTORY(ies), if they are empty.  --ignore-fail-on-non-empty ignore each
failure to remove a non-empty directory -p, --parents remove DIRECTORY and its
ancestors; e.g., 'rmdir -p a/b' is similar to 'rmdir a/b a' -v, --verbose
output a diagnostic for every directory processed --help display this help and
exit --version output version information and exit Written by David MacKenzie.
