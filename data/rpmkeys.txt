rpmkeys - RPM Keyring rpmkeys {-K|--checksig} [options] PACKAGE_FILE ...
rpmkeys {-d|--delete|-e|--erase} [options] FINGERPRINT ... rpmkeys
{-x|--export} [options] [FINGERPRINT ...] rpmkeys {-i|--import} [options]
PUBKEY ... rpmkeys {-l|--list} [options] [FINGERPRINT ...] rpmkeys --rebuild
[options] [rebuild-options] rpmkeys is used for manipulating the rpm keyring
and verifying package digital signatures with the contained keys. For all
available operations, see OPERATIONS. -K, --checksig Verify the digests and
signatures contained in PACKAGE_FILE to ensure the integrity and origin of the
package. -d, --delete, -e, --erase Erase the key(s) designated by FINGERPRINT.
The --delete and -d options are deprecated. -x, --export Output the key(s)
designated by FINGERPRINT using an ASCII-armor encoding.  If FINGERPRINT is
not specified, output all keys. --import Import ASCII-armored public keys.
Digital signatures cannot be verified without the corresponding public key
(aka certificate). -l, --list List currently imported public key(s) (aka
certificates) by their fingerprint and user ID. If no fingerprints are
specified, list all keys. --rebuild Recreate the public key storage. Update to
the latest format and drop unreadable keys. FINGERPRINT The handle used for
all operations on the keys. PACKAGE_FILE An rpm package file or a manifest.
PUBKEY An ASCII-armored OpenPGP public key (aka certificate). See rpm-
common(8) for the options common to all rpm executables. --from
<fs|openpgp|rpmdb> Use the keys from the specified backend to rebuild the
currently configured keystore backend. This can be used to convert from one
key storage to another. --checksig
<_PACKAGE_FILE_>: <element> <element> <OK|NOT OK>
With --verbose:
<_PACKAGE_FILE_>:
    <element>: <OK|NOTFOUND|BAD>
    ...
--list
<fingerprint> <name> <userid> public key
There are several configurables affecting the behavior of this verification,
see rpm-config(5) for details: %_keyring %_keyringpath %_pkgverify_flags
%_pkgverify_level On success, 0 is returned, a nonzero failure code otherwise.
