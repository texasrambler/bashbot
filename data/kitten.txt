kitten - Fast, statically compiled implementations of various kittens (command
line tools for use with kitty) kitten serves as a launcher for running
individual kittens. Each kitten can be run as kitten command. The list of
available kittens is given below.  @ Control kitty remotely. See: ask Ask the
user for input. See: choose-files Choose files, fast. See: choose-fonts Choose
the fonts used in kitty. See: clipboard Copy/paste with the system clipboard,
even over SSH. See: desktop-ui Implement various desktop components for use
with lightweight compositors/window managers on Linux. See: diff Pretty,
side-by-side diffing of files and images. See: edit-in-kitty Edit a file in a
kitty overlay window. See: hints Select text from screen with keyboard. See:
hyperlinked-grep Add hyperlinks to the output of ripgrep. See: icat Display
images in the terminal. See: mouse-demo Demo the mouse handling kitty
implements for terminal programs. See: notify Send notifications to the user.
See: panel Use a command line program to draw a GPU accelerated panel on your
desktop. See: query-terminal Query the terminal for various capabilities. See:
quick-access-terminal A quick access terminal window that you can bring up
instantly with a keypress or a command.. See: run-shell Run the user's shell
with shell integration enabled. See: show-key Show the codes generated by the
terminal for key presses in various keyboard modes. See: ssh Truly convenient
SSH. See: themes Manage kitty color schemes easily. See: transfer Transfer
files easily over the TTY device. See: unicode-input Browse and select unicode
characters by name. See: update-self Update this kitten binary. See: Get help
for an individual command by running: command -h The current kitten version.
