autoupdate - Update a configure.ac to a newer Autoconf [OPTION]... [TEMPLATE-
FILE]...  Update each TEMPLATE-FILE if given, or 'configure.ac' if present, or
else 'configure.in', to the syntax of the current version of Autoconf.	The
original files are backed up.  -h, --help print this help, then exit -V,
--version print version number, then exit -v, --verbose verbosely report
processing -d, --debug don't remove temporary files -f, --force consider all
files obsolete -B, --prepend-include=DIR prepend directory DIR to search path
-I, --include=DIR append directory DIR to search path Written by David J.
