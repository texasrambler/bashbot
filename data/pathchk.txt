pathchk - check whether file names are valid or portable [OPTION]... NAME...
Diagnose invalid or non-portable file names.  -p check for most POSIX systems
-P check for empty names and leading "-" --portability check for all POSIX
systems (equivalent to -p -P) --help display this help and exit --version
output version information and exit Written by Paul Eggert, David MacKenzie,
