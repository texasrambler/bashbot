systemd-hwdb - hardware database management tool systemd-hwdb [options] update
systemd-hwdb [options] query modalias systemd-hwdb expects a command and
command specific arguments. It manages the binary hardware database.  --usr
Generate in /usr/lib/udev instead of /etc/udev.

Added in version 219.  -r, --root=PATH Alternate root path in the filesystem.

Added in version 219.  -s, --strict When updating, return non-zero exit value
on any parsing error.

Added in version 239.  -h, --help Print a short help text and exit.  Update
the binary database.  Query database and print result.	systemd(1) hwdb(7)
