infotocap - convert a terminfo description into a termcap description ...
infotocap translates terminal descriptions.  It looks in each given text file
for terminfo entries and, for each one found, it writes an analogous termcap
description to the standard output stream.  terminfo “use” capabilities
translate to termcap tc capabilities.  Because termcap is a less expressive
format than terminfo, some capabilities cannot be translated.  This utility is
implemented as a link to tic(1M), with the latter's option implied.  You can
use other tic options such as and The -V option reports the version of ncurses
associated with this program and exits with a successful status.  compiled
