Print version and exit.	 Show summary of options.  Log output to FILE. OBJECTS
is the list of object types that we want to monitor. It may contain 'link',
'address', 'route' and 'all'. 'link' specifies the network device, 'address'
the protocol (IP or IPv6) address on a device, 'route' the routing table entry
and 'all' does what the name says.  Specify protocol family. 'inet' is IPv4,
'inet6' is IPv6, 'link' means that no networking protocol is involved and
'help' prints usage information.  Use IPv4. Shortcut for -family inet.	Use
IPv6. Shortcut for -family inet6.  Use a special family identifier meaning
that no networking protocol is involved. Shortcut for -family link.  Log to
file /var/log/rtmon.log, then run: to display logged output from file.	was
