colrm - remove columns from a file

colrm [first [last]]

colrm removes selected columns from a file. Input is taken from standard
input. Output is sent to standard output.

If called with one parameter the columns of each line will be removed starting
with the specified first column. If called with two parameters the columns
from the first column to the last column will be removed.

Column numbering starts with column 1.

-h, --help Display help text and exit.

-V, --version Display version and exit.

The colrm command appeared in 3.0BSD.
