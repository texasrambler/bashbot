deallocvt - deallocate unused virtual consoles [option...]  The command
deallocates kernel memory and data structures for all unused virtual consoles.
If one or more arguments are given, only the corresponding consoles are
deallocated.

A virtual console is unused if it is not the foreground console, and no
process has it open for reading or writing, and no text has been selected on
its screen.  -C, --console=DEV the console device to be used.  -V, --version
print version number.  -h, --help print this usage message.  Andries Brouwer
<aeb@cwi.nl>
