setvtrgb - set the virtual terminal RGB colors [options] The command takes a
single argument, either the string , or a path to a file containing the colors
to be used by the Linux virtual terminals.

You can choose to write the colors in decimal or hexadecimal format, it will
be detected on runtime.

Decimal format should be exactly 3 lines of 16 comma-separated decimal values
for RED, GREEN, and BLUE.
