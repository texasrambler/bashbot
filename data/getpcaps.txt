getpcaps - display process capabilities displays the capabilities on the
processes indicated by the value(s) given on the command line.	A of 0
displays the capabilities of the process that is running itself.  The
capability Set of the process is displayed in the text represenation described
here: Optional arguments: Displays usage information and exits.	 Displays
output in a somewhat ugly legacy format.  Displays usage in a legacy-like
format but not quite so ugly in modern default terminal fonts.	Displays IAB
tuple capabilities from the process. The output format here is also described
