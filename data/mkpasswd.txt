the libc function, using the given salt.  Use the as salt.  If it begins with
then it will be passed straight to without any checks.	Use rounds.  This
argument is ignored if the method chosen does not support variable rounds.
For the OpenBSD Blowfish method this is the logarithm of the number of rounds.
The behavior is undefined if this option is used without Compute the password
using the method.  If is then the list of available methods is printed.	 If
begins and end with characters then the string is passed to as-is.  Like Read
the password from file descriptor instead of using If the file descriptor is
not connected to a tty then no other text than the hashed password is printed
on stdout.  Like A list of options which will be evaluated before the ones
specified on the command line.	If the option is used then passwords
